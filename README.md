# PerformIQ: Employee Performance Tracking System 🚀

PerformIQ is a premium MERN stack application for tracking and managing employee performance with a modern business aesthetic.

## 🏗️ Project Structure
- `/client`: React (Vite) frontend.
- `/server`: Node.js (Express) backend.

## 🚀 Getting Started
1. Clone the repo.
2. Install dependencies in both folders (`npm install`).
3. Set up your `.env` in `/server`.
4. Run `npm run dev` in both folders.
